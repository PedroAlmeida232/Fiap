React é framework baseado em javascript
criado por Jordan Walke em 2011 e lançamento pelo facebook foi 2013

Virtual DOM (Document Object Model Virtual): Em vez de atualizar o DOM real diretamente a cada mudança, 
o React cria uma representação virtual do DOM. Ele compara a versão virtual atual com a anterior e aplica apenas 
as mudanças necessárias ao DOM real. Isso minimiza as operações pesadas no navegador, resultando em interfaces mais 
rápidas e responsivas.

Componentes: O React promove uma arquitetura baseada em componentes, onde a UI é dividida em pequenas partes
independentes e reutilizáveis. Essa modularidade facilita o desenvolvimento, 
a manutenção e a escalabilidade das aplicações.

componentes = páginas
renderização = pagina index.html consumindo um ou vários componentes
html + javascript = jsx, tsx
